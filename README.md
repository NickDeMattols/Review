# Comandos do Git Bash:

LS = listar
CD = locomover nas pastas
mkdir = criar pastas
code . = abre o Vscode
git init = cria um repositório (master)
git add . = seleciona tudo da pasta pra enviar
git commit -m "comentário" = preparar o arquivo para upload
git config --global user.email "..."
git config --global user.name "..."
git branch -M main (main)
git remote add origin "link do arquivo"
git push "-u origin main" (caso seja a primeira vez) = envia o arquivo
git pull = atualizar repositório
touch = criar um ou mais arquivos
